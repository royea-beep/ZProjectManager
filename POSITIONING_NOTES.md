## KeyDrop

**Current tagline:** "Stop getting API keys over WhatsApp"
**Verdict:** Excellent. Keep it. It's specific, relatable, and immediately communicates the problem.

**Positioning:** Security tool for freelancers and agencies. Not enterprise security — lightweight, fast, no-BS credential collection.

**Key differentiators:**
- One-time links (expire after use)
- AES-256 encryption (not just HTTPS)
- 30+ pre-built templates (Stripe, AWS, Firebase, etc.)
- Audit trail on every action
- Hebrew + English (RTL-aware)

**Avoid:** Don't position as "password manager" (1Password/Bitwarden own that). Position as "credential handoff tool."

## SecretSauce

**Current tagline:** "Find the business logic you forgot to protect"
**Verdict:** Strong. Clear problem statement.

**Positioning:** Developer security tool. Not a linter, not a vulnerability scanner — a **business logic exposure detector**. Unique category.

**Key differentiators:**
- No competition (literally the only tool in this niche)
- Zero dependencies (runs anywhere)
- Actionable output (tells you WHERE to move code, not just what's wrong)
- Star rating (prioritizes findings by uniqueness)
- Protection levels with cache TTL recommendations

**Pricing position:** Premium one-time ($49-79). Not subscription. Developers buy tools, not subscriptions for tools they use monthly.

**Future upsell:** Pro tier ($199) for CI/CD integration, team reports, custom pattern definitions.

## VenueKit

**Positioning:** White-label website builder for gaming/sports venues. Not a website builder (Wix/Squarespace own that). A **venue-specific, ready-to-deploy solution**.

**Key differentiators:**
- Fully built site in minutes (not "build your own")
- Admin panel included (tournament management, not just a brochure)
- Hebrew + RTL out of the box
- Supabase backend (real database, not just static pages)
- PWA-ready (players install as app)
- Live proof: heroes.ftable.co.il

**Sales angle for Israel market:**
> "אתר מלא לקלאב שלך ב-₪2,500 — כולל ניהול טורנירים, ליגה, גלריה, ופאנל ניהול. מוכן תוך יום."
