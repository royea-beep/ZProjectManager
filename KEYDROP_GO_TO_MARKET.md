**Updated:** 2026-03-09

## Product

**KeyDrop** — Secure credential collection through encrypted, one-time links.

**Tagline:** "Stop getting API keys over WhatsApp."

**Problem:** Developers collect API keys, passwords, and credentials from clients through insecure channels: WhatsApp, email, Slack. No encryption, no audit trail, no expiry.

**Solution:** Create a request with pre-built templates (Stripe, AWS, Firebase, etc.), share a one-time link, client fills in the form, credentials are encrypted with AES-256-GCM and stored securely.

---

## Target Audience (in order)

1. **Freelance developers** — Collect client credentials for projects (Stripe keys, Firebase configs, API tokens)
2. **Small agencies** — Manage credentials across multiple client projects
3. **DevOps/IT** — Collect infrastructure credentials from team members
4. **Consultants** — Gather access credentials during onboarding
